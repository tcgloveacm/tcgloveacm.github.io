2019 述职报告

发表于 2019-04-30 | 分类于 人生

述职经历

述职终于结束，忐忑的心终于可以放下来好好工作了。因为是第一次做这种性质报告，准备 PPT 搞得焦头烂额，抱着电脑在咖啡厅绞尽脑汁冥思苦想：

我是谁？
我干了什么？
我未来要干什么？

作为一个普通员工，特别是业务工程师应该都有一种感觉：每天都很忙，一回想又感觉什么都没做，总不能把实现了某个功能点写在 PPT 上，就算实现了某个功能也不是一个人的功劳。🙃不过，苦思冥想几天还是有点心得的，也是一种人生的宝贵经历吧。

经验总结

报告书写

报告内容切忌流水账式：我做了 A，做了 B，做了 CDEFG…，没人关注你做了啥，关键在于：做了什么 -> 产生了什么结果。然后进行报告的时候，顺序颠倒下：为了产生什么结果 <- 我都做了什么。建议参考：金字塔原理，以结果为导向的论述过程。

金字塔原理四个特征

  1. 结论先行
  2. 以上统下
  3. 归类分组
  4. 逻辑递进

PPT 演讲

我认为 PPT 是用来概述你的演讲脉络和抓住听众的工具。所以制作 PPT 切忌大段大段的文字，多采用形象化图表。只要按照金字塔原理展示主题和关键数据图表即可。

汇报的时候有一个小技巧，举个形象生动的例子：
屡战屡败
屡败屡战

不足

性格缺点：太被动，缺乏主动性
这种性格不仅仅体现在工作上。曾经过往的人和事，如果当初能够积极主动一点点，也许就不会错过吧。

站在二十岁的尾巴上，遥望而立之年，送自己一句话吧：知其不可而为止。
